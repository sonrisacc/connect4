## Initial Setup

1. Fork and clone the repo
2. Install dependencies from the root of the repo by running

```sh
npm install
```

3. Start connectFour in dev mode by running:

```sh
npm run build
npm run dev
```

or Start connectFour by running:

```sh
npm start
```

4. Run test by running:

```sh
npm test
```
